- Backup your current config:
```console
# required
mv ~/.config/nvim{,.bak}

# optional but recommended
mv ~/.local/share/nvim{,.bak}
mv ~/.local/state/nvim{,.bak}
mv ~/.cache/nvim{,.bak}
```
- Clone the repo:
```console
git clone https://github.com/Roconx/LazyVim-config.git ~/.config/nvim
```

- Start Neovim:
```console
nvim
```
